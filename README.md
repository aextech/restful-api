# 错误码
错误码 | 说明
-----  | ---------
0      | 正常
1001   | 系统错误
4001   | 无效请求
5001   | 未认证
6001   | 已认证
7001   | 系统繁忙
8001   | 计价币不存在
9001   | 交易币不存在
10001  | 无效交易区
11001  | 该币在当前交易区不可交易
